**接口文档**

1. 查看测试样例

```http
GET /api/examples/payments/demo
```

2. 创建订单

```http
POST /api/examples/payments/orders
Content-Type: application/json
```

```json
{
  "orderNo": "PAY_NEW_20260320_001",
  "idempotencyKey": "IDEMP_NEW_20260320_001",
  "userId": 1001,
  "productCode": "VIP_MONTH",
  "amountFen": 9900
}
```

3. 发起支付

```http
POST /api/examples/payments/orders/PAY_NEW_20260320_001/start
Content-Type: application/json
```

```json
{
  "idempotencyKey": "IDEMP_NEW_20260320_001"
}
```

4. 模拟支付成功回调

```http
POST /api/examples/payments/orders/PAY_NEW_20260320_001/success
Content-Type: application/json
```

```json
{
  "channelTradeNo": "WX202603200099",
  "paidAt": "2026-03-20T16:30:00"
}
```

5. 查询订单

```http
GET /api/examples/payments/orders/PAY_NEW_20260320_001
```

6. 关闭订单

```http
POST /api/examples/payments/orders/PAY_NEW_20260320_001/close
```

**推荐测试顺序**

1. 先查 demo

```bash
curl http://localhost:8924/api/examples/payments/demo
```

2. 创建一笔新订单

```bash
curl -X POST http://localhost:8924/api/examples/payments/orders \
  -H "Content-Type: application/json" \
  -d "{\"orderNo\":\"PAY_NEW_20260320_001\",\"idempotencyKey\":\"IDEMP_NEW_20260320_001\",\"userId\":1001,\"productCode\":\"VIP_MONTH\",\"amountFen\":9900}"
```

3. 同样请求再发一次，验证幂等创建

```bash
curl -X POST http://localhost:8924/api/examples/payments/orders \
  -H "Content-Type: application/json" \
  -d "{\"orderNo\":\"PAY_NEW_20260320_001\",\"idempotencyKey\":\"IDEMP_NEW_20260320_001\",\"userId\":1001,\"productCode\":\"VIP_MONTH\",\"amountFen\":9900}"
```

4. 发起支付

```bash
curl -X POST http://localhost:8924/api/examples/payments/orders/PAY_NEW_20260320_001/start \
  -H "Content-Type: application/json" \
  -d "{\"idempotencyKey\":\"IDEMP_NEW_20260320_001\"}"
```

5. 再发一次发起支付，验证“支付中”幂等

```bash
curl -X POST http://localhost:8924/api/examples/payments/orders/PAY_NEW_20260320_001/start \
  -H "Content-Type: application/json" \
  -d "{\"idempotencyKey\":\"IDEMP_NEW_20260320_001\"}"
```

6. 回写成功

```bash
curl -X POST http://localhost:8924/api/examples/payments/orders/PAY_NEW_20260320_001/success \
  -H "Content-Type: application/json" \
  -d "{\"channelTradeNo\":\"WX202603200099\",\"paidAt\":\"2026-03-20T16:30:00\"}"
```

7. 再回写一次成功，验证成功幂等

```bash
curl -X POST http://localhost:8924/api/examples/payments/orders/PAY_NEW_20260320_001/success \
  -H "Content-Type: application/json" \
  -d "{\"channelTradeNo\":\"WX202603200099\",\"paidAt\":\"2026-03-20T16:30:00\"}"
```

8. 查询最终状态

```bash
curl http://localhost:8924/api/examples/payments/orders/PAY_NEW_20260320_001
```
